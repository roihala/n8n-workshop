Lesson 1 Mastering n8n: Data, Context, and LLM-Powered Building

Duration: 90 minutes (45 min talk + 45 min practice)

By the end of this lesson, students will be able to: build multi-step workflows with proper data flow, use an LLM to generate and iterate on complete n8n workflows, apply prompt engineering best practices, debug workflows using LLMs, and handle errors gracefully.

1.4 Error Handling5 minError workflows, retry logic, fallback paths

# Verify n8n is running

curl -s "https://generativelanguage.googleapis.com/v1/models?key=$GEMINI_API_KEY" | head -1

# Verify API key access (Grok / xAI)
curl -s https://api.x.ai/v1/models -H "Authorization: Bearer $XAI_API_KEY" | head -1
